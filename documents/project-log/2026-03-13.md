# 2026-03-13 개발 기록

## 🛠️ 작업 내용
* **지능형 메뉴 옵션 자동 매핑**:
  - 메뉴명에 '라떼', '프라푸치노' 등 특정 키워드가 포함될 시 '휘핑유무' 등의 관련 옵션 그룹을 자동으로 매핑해주는 스마트 서비스 로직 구축.
* **주문 취소 및 결제 원자성 확보 (Point Rollback)**:
  - `CANCELLED` 처리 시 상태 전이와 포인트 회수/환불을 일괄 트랜잭션으로 제어하여 데이터 모순을 차단.
  - 주문 시 사용한 포인트는 복구 환불 처리하고, 주문으로 적립된 보너스 포인트는 자동으로 회수.
* **관리자 조작 안전성 보완**:
  - 주문 목록 조회 시 회원 고유키 대신 전화번호를 조인 출력하여 식별성 제공.
  - 모든 조작 창(수정, 삭제 등)에 '취소(0)' 옵션을 추가하여 데이터 소실 방지.
* **데이터 모델 고도화**:
  - `CATEGORY_OPTION_GROUP` 테이블 도입을 통한 카테고리별 공통 옵션 일괄 매핑 체계 정비.
  - `Category` 모델 내에 `List<OptionGroup>` 을 상속 구조로 정의하여 객체 지향적 도메인 설계.
