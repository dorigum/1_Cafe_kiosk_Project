# 2026-03-12 개발 기록

## 🛠️ 작업 내용
* **MVC 패턴 정문화 및 Controller 계층 도입**:
  - 기존에 혼재되어 있던 흐름 제어를 `AdminController`로 일원화하고, View 계층을 정적 호출 방식으로 완전 격리.
  - `FailView` 및 `EndView` 의 데이터 바인딩 로직을 개선하여 레이어 간 상호 의존성 최소화.
* **데이터 무결성 및 제약 조건 강화**:
  - `CATEGORY` 테이블의 `category_name` 및 `MENU` 테이블의 `menu_name`에 `UNIQUE` 제약 조건을 추가하여 중복 명칭 유입 차단.
* **매출 통계 기반 다차원 기간 분석**:
  - 날짜 입력 범위를 기준으로 하여 일별, 주별, 월별, 연도별 분석을 제공하고 주차별 데이터 표현 포맷('YYYY-MM-W주') 적용.
